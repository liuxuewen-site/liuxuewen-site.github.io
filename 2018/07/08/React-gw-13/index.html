3、首字母大写：

当元素标签以小写字母开头时，它表示一个内置的组件，如 < div > 或 < span >，并将字符串 ‘div’ 或 ‘span’ 传 递给 React.createElement 。以大写字母开头的标签，如 < Foo /> 会被编译成 React.createElement(Foo) ，并它正对应于你在 JavaScript 文件中定义或导入的组件。

我们建议用大写开头命名组件。如果你的组件以小写字母开头，请在 JSX 中使用之前其赋值给大写开头的变量。
