它可以正常运行，因为在 JavaScript 中， true && expression 会评估为 expression ，而 false && expression 执行为 false 。因此，如果条件为 true ，则 && 后面的元素将显示在输出中。 如果是 false，React 将会忽略并跳过它。

在极少数情况下，可能希望组件隐藏自身，即使它被其他组件渲染。让 render 方法返回 null 而不是它的渲染结果即可实现。
